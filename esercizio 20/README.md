# Esercizio 20
In un'app di ricette, hai gli ingredienti in un array; stampali in modo leggibile nella descrizione del piatto

## Codifica
``` js
const ingredienti = ["Farina", "Uova", "Latte", "Zucchero", "Estratto di vaniglia", "Gocciole di cioccolato", "Burro", "Lievito"];
const lista = ingredienti.join(",");

console.log(lista);
```
