# Esercizio 24
In un magazzino, gli ultimi scatoloni arrivati sono i primi ad essere spediti. Il magazziniere spedisce un pacco, stampa la lista aggiornata

## Codifica
``` js
const magazzino = ["Frutta", "Verdura", "Pane", "Utensili", "Prodotti congelati"];
magazzino.pop();

console.log(magazzino);
```
